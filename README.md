## 概要
#### ポートフォリオサイトです
 Reactで作った、シンプルなデザイン、シンプルな動きのポートフォリオサイトです。
 デザインに関しては、各モーダルそれぞれを直感的に認識できればと、背景色の色分けをしました。
 動きは、React Transition GroupとCSSで実装し、リロードに関しては、React RouterのHashRouterで対応しました。
