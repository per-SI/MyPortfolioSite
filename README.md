## 概要
#### ポートフォリオサイト
　Reactで作った、シンプルなデザイン、シンプルな動きのポートフォリオサイトです。
 
　デザインは、各モーダルそれぞれを直感的に認識できればと、背景色の色分けをしました。
 
　動きに関しては、React Transition GroupとCSSで実装し、リロードに関しては、React RouterのHashRouterで対応しています。
  
　レスポンシブ対応◎
